Pill label for type tags (`COURSE`/`WORKSPACE`), statuses and counts. `course` is blue, `workspace` amber — the two workspace modes.

```jsx
<Badge tone="course" uppercase>Course</Badge>
<Badge tone="workspace" uppercase>Workspace</Badge>
<Badge tone="success">8/10</Badge>
<Badge tone="purple">In progress</Badge>
```

Tones: neutral, course, workspace, success, info, warning, error, purple, green, dark. `uppercase` for type tags, `size="sm"` for dense rows.
