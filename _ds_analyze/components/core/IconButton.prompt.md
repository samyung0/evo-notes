Square icon-only button — used for search (`dark`), notifications (`accent` + `dot`), overflow menus (`ghost`), and toolbar actions.

```jsx
<IconButton icon="search" variant="dark" />
<IconButton icon="bell" variant="accent" dot />
<IconButton icon="plus" variant="outline" />
```

Sizes `sm`/`md`/`lg`. `dot` adds a red unread indicator. Variants: `dark`, `accent` (purple fill, white icon), `outline`, `ghost`.
