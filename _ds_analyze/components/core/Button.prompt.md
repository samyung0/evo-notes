Text button for primary and secondary actions — `primary` (near-black) is the default CTA; `accent` is the purple fill (white text); `outline`/`ghost` step down in emphasis.

```jsx
<Button variant="primary" iconLeft="plus">New course</Button>
<Button variant="accent" iconLeft="sparkles">Generate quiz</Button>
<Button variant="outline" size="sm">Open</Button>
```

Sizes: `sm`, `md` (default), `lg`. Add `iconLeft`/`iconRight` (Icon names), `fullWidth`, or `disabled`.
