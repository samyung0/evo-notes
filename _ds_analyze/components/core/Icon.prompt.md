Renders a single line icon from the Evo Notes set — use anywhere the UI needs an icon; pairs with `Button`, `IconButton`, and nav rows.

```jsx
<Icon name="sparkles" size={18} />
<Icon name="bell" size={17} color="var(--ev-purple-ink)" />
```

Names: dashboard, workspaces, practice, schedule, files, tasks, notes, profile, settings, logout, search, bell, plus, minus, chevronDown, chevronRight, chevronLeft, arrowRight, upload, send, sparkles, check, more, message, book, flashcards, quiz, x, filter, clock. Inherits `currentColor` by default, so set color via the parent's `color`.
